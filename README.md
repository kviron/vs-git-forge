## Работа с Markdown

README можно редактировать в Visual Studio Code. Полезные сочетания клавиш:

* Разделить редактор (`Cmd+\` в macOS или `Ctrl+\` в Windows и Linux).
* Включить предпросмотр (`Shift+Cmd+V` в macOS или `Shift+Ctrl+V` в Windows и Linux).
* Нажми `Ctrl+Пробел` (Windows, Linux, macOS), чтобы увидеть список сниппетов Markdown.
